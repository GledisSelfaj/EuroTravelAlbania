**EuroTravel Albania** is a web platform created to present a bus transportation service that offers travel from Albania to different destinations across Europe. The main goal of the project is to provide users with clear information about bus routes, destinations, travel schedules, and ticket booking options. The website is designed to be simple and user-friendly so that travelers can easily find the information they need for their trips.

Technologies Used:

HTML – for the structure of the website

CSS – for styling and layout

JavaScript – for interactive features

Bootstrap – for responsive design and ready-made components
